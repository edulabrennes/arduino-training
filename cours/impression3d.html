Fonctionnement électronique

  • Eléctronique propriétaire / open source

  • Cas d’une arduino + ramps

  • Upload d’un firmware (7 min 10s)

  • Configuration du firmware Marlin

  • Pour éviter de réuploader à chaque fois, l’EEPROM

Fonctionnement logiciel

  • Conception d’un modèle 3D avec un modeleur (sketchup, Tinkercad, Freecad)

  • Enregistrer au format STL

  • Passage dans un slicer (Cura)

  • Enregistrer au format GCODE

  • Passage dans l’imprimante 3D

Le slicer Cura

  • Charger le STL sur thingiverse

  • Bouger, zoomer, tourner le modèle

  • Agrandir / rétrécir

  • Mirroir

  • Dupliquer
