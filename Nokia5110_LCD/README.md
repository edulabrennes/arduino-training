# Nokia5110 LCD

Pour afficher un "hello world"

## Branchements

SCLK    D13
DIN     D11
DC      D5
CE      D7
RST     D6
VCC     5v
BL      3v3
GND     GND

## Librairies

Pour installer les librairies : 

Croquis > Inclure une bibliothèque > Gérer les bibliothèque

puis installer :
 - Adafruit GFX 
 - Adafruit PCD8544 Nokia-5110-LCD
